# WAVE — Marketing Operations Platform

> Third pillar of the LYC Partners platform stack alongside **VISTA** (BD Command Center) and **DEX AI** (Intelligent ATS).

## Overview

WAVE manages the entire demand generation engine — from content planning through asset generation, distribution, registration, and B2C inbound journey orchestration.

**Core capabilities:**
- Content Command Center (editorial calendar, campaign planning)
- Template & Asset Library (10+ template types, AI-assisted generation)
- Distribution Engine (email sequences, mailing lists, scheduling)
- B2C Journey Engine (automated funnel, zero human sales)
- Content Repurposing Engine (1 webinar → 10+ derivatives)
- Registration & Event Management (unified registration, payments)
- Analytics & Intelligence (content performance, campaign ROI)
